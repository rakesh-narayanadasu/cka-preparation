# CKA Preparation

This repository contains my personal notes, practice exercises, and resources for preparing for the **Certified Kubernetes Administrator (CKA)** exam.

The goal is to keep everything **hands-on**, **exam-focused**, and easy to review.

---

## Exam Overview

The **CKA (Certified Kubernetes Administrator)** exam tests practical skills related to:
- Kubernetes cluster administration
- Troubleshooting
- Configuration and management of workloads
- Networking, storage, and security

The exam is **performance-based**, meaning you solve real Kubernetes tasks in a live environment.

---
## Topics Covered

- **Core Concepts**
  - Kubernetes architecture
  - API server, etcd, scheduler, controller manager

- **Workloads & Scheduling**
  - Pods, Deployments, Jobs, CronJobs
  - Labels, selectors, taints, and tolerations

- **Services & Networking**
  - Services, Ingress
  - DNS and networking basics

- **Storage**
  - Volumes
  - PersistentVolumes (PV) and PersistentVolumeClaims (PVC)

- **Security**
  - RBAC
  - ServiceAccounts
  - Security contexts

- **Cluster Maintenance**
  - Upgrades
  - Backup and restore
  - Node management

- **Troubleshooting**
  - Debugging pods and nodes
  - Logs and events
